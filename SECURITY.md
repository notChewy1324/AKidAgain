# Security Policy

## Supported Versions

Newer versions will be supported while older version get phased out.

| Version | Supported          |
| ------- | ------------------ |
| >= 5.0   | :white_check_mark: |
| 4.0 - 4.3| :white_check_mark:  |
| 3.0     | :x:| 
| <= 2.8   | :x:                |

## Reporting a Vulnerability

Please file an issue on github for review.
